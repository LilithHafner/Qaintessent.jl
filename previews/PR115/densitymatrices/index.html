Density Matrix representation

Qaintessent.jl supports the manipulation of mixed states as DensityMatrix(s).

Density Matrices

Qaintessent.DensityMatrix — Type
DensityMatrix

Density matrix, represented with respect to identity and Pauli basis (σ_j/2) by storing 4 real coefficients for each qubit, i.e., a real vector of length 4^N for N qubits.
