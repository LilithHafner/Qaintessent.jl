Gates

Any quantum operation can be represented as a unitary matrix, which are referred to as gates. In Qaintessent.jl, basic gates are implemented using the abstract base struct AbstractGate. The available basic quantum gates are seen below. Note that wire index information is still needed to when applying such gates. As such, these gates are permuted when creating the basic building blocks of Qaintessent.jl Quantum Circuit(s): CircuitGate(s).

Qaintessent.AbstractGate — Type
AbstractGate

Abtract unitary quantum gate. N is the number of "wires" the gate acts on.

Pauli X gate

Qaintessent.YGate — Type

Pauli Y gate

$Y = \begin{pmatrix} 0 & -i \\ i & 0 \end{pmatrix}$

Qaintessent.ZGate — Type

Pauli Z gate

$Z = \begin{pmatrix} 1 & 0 \\ 0 & -1 \end{pmatrix}$

source
Qaintessent.HadamardGate — Type

Hadamard gate

$H = \frac{\sqrt} \begin{pmatrix} 1 & 1 \\ 1 & 1 \end{pmatrix}$

source
Qaintessent.SGate — Type

S gate

$S = \frac{\sqrt} \begin{pmatrix} 1 & 0 \\ 0 & i \end{pmatrix}$

S† gate

$S^{†} = \begin{pmatrix} 1 & 0 \\ 0 & -i \end{pmatrix}$

source
Qaintessent.TGate — Type

T gate

$T = \frac{\sqrt} \begin{pmatrix} 1 & 0 \\ 0 & e^{\frac{iπ}{4}} \end{pmatrix}$

T† gate

$T^{†} = \begin{pmatrix} 1 & 0 \\ 0 & e^{-\frac{iπ}{4}} \end{pmatrix}$

Parametrized Single Qubit Gates

Qaintessent.RxGate — Type

Rotation-X gate

$R_{x}(\theta) = \begin{pmatrix} \cos(\frac{\theta}) & -i\sin(\frac{\theta}) \\ -i\sin(\frac{\theta}) & \cos(\frac{\theta}) \end{pmatrix}$

source
Qaintessent.RyGate — Type

Rotation-Y gate

$R_{y}(\theta) = \begin{pmatrix} \cos(\frac{\theta}) & -\sin(\frac{\theta}) \\ \sin(\frac{\theta}) & \cos(\frac{\theta}) \end{pmatrix}$

source
Qaintessent.RzGate — Type

Rotation-Z gate

$R_{z}(\theta) = \begin{pmatrix} e^{\frac{-i\theta}} & 0 \\ 0 & e^{\frac{i\theta}} \end{pmatrix}$

source
Qaintessent.RotationGate — Type

General rotation operator gate: rotation by angle θ around unit vector n.

$R_{\vec}(\theta) = \cos(\frac{\theta})I - i\sin(\frac{\theta})\vec\sigma, \\ \sigma = [X, Y, Z]$

source
Qaintessent.PhaseShiftGate — Type

Phase shift gate

$P(\phi) = \begin{pmatrix} 1 & 0 \\ 0 & e^{i\phi} \end{pmatrix}$

source

Qaintessent.SwapGate — Type

Swap gate

Quantum Gate Matrix Representation

The Qaintessent.matrix function can be used to convert AbstractGate objects to their matrix representation. 

using Qaintessent

x = XGate()
Qaintessent.matrix(x)
2×2 Array{Complex{Float64},2}:
 0.0+0.0im  1.0+0.0im
 1.0+0.0im  0.0+0.0im
