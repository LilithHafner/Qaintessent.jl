Gates

Any quantum operation can be represented as a unitary matrix, which are referred to as gates. In Qaintessent.jl, basic gates are implemented using the abstract struct AbstractGate. The available basic quantum gates are seen below. Note that at this level of abstraction, wire index information is ignored and matrixes are permuted such that the fastest running qubit is also the fastest running index.

These gates are permuted when creating the basic building blocks of Qaintessent.jl Quantum Circuit(s): CircuitGate(s).

Qaintessent.AbstractGate — Type
AbstractGate

Abtract unitary quantum gate. N is the number of "wires" the gate acts on.

Parametrized Gates

Qaintessent.RxGate — Type
Rotation-X gate

$R_{x}(\theta) = \begin{pmatrix} \cos(\frac{\theta}{2}) & -i\sin(\frac{\theta}{2}) \\ -i\sin(\frac{\theta}{2}) & \cos(\frac{\theta}{2}) \end{pmatrix}$

source
Qaintessent.RyGate — Type
Rotation-Y gate

$R_{y}(\theta) = \begin{pmatrix} \cos(\frac{\theta}{2}) & -\sin(\frac{\theta}{2}) \\ \sin(\frac{\theta}{2}) & \cos(\frac{\theta}{2}) \end{pmatrix}$

source
Qaintessent.RzGate — Type
Rotation-Z gate

$R_{z}(\theta) = \begin{pmatrix} e^{\frac{-i\theta}{2}} & 0 \\ 0 & e^{\frac{i\theta}{2}} \end{pmatrix}$

source
Qaintessent.RotationGate — Type
General rotation operator gate: rotation by angle `θ` around unit vector `n`.

$R_{\vec v}(\theta) = \cos(\frac{\theta}{2})I - i\sin(\frac{\theta}{2})\vec v \sigma, \\ \sigma = [X, Y, Z]$

source
Qaintessent.PhaseShiftGate — Type
Phase shift gate

$P(\phi) = \begin{pmatrix} 1 & 0 \\ 0 & e^{i\phi} \end{pmatrix}$

source
Qaintessent.EntanglementXXGate — Type
Entanglement-XX gate

$G_{x}(\theta) = e^{-i \theta X \otimes X / 2}$
