Circuit Construction and Usage

The output of typical Quantum simulation is the expectation values of the output quantum state based on certain measurement operators. In Qaintessent.jl, this is accomplished by applying a Quantum Circuit(s) consisting of a CircuitGateChain and Measurement Operators. See the corresponding sections for more information.

CircuitGate

The basic building block of any CircuitGateChain is CircuitGates. These are constructed from basic quantum Gates. The constructor for CircuitGates is complicated: CircuitGate Helper Functions and a CircuitGate Example can be found below.

CircuitGate Example

The Qaintessent.matrix function can be used to convert CircuitGate objects to a CSC sparse matrix representation

    using Qaintessent
    N = 2
    cnot = controlled_circuit_gate(1, 2, XGate(), N)

    Qaintessent.matrix(cnot)
4×4 SparseArrays.SparseMatrixCSC{Float64,Int64} with 4 stored entries:
  [1, 1]  =  1.0
  [2, 2]  =  1.0
  [4, 3]  =  1.0
  [3, 4]  =  1.0

The CircuitGate can then be applied to a quantum state in state vector form.

    ψ = [0, 0, 1, 0]

    apply(cnot, ψ)
4-element Array{Int64,1}:
 0
 0
 0
 1

Moments

When performing a quantum simulation, it may be required to define an intermediate state; this can be used to optimize the mapping of gates to physical qubits. In Qaintessent.jl, we allow for groupings of CircuitGate(s) that can be run in parallel into Moments. A Moment Example is shown below.

Moment Example

    using Qaintessent

    N = 3
    m = Moment{N}(
        [
            controlled_circuit_gate(1, 2, XGate(), N),
            single_qubit_circuit_gate(3, RxGate(0.2π), N),
        ]
    )

CircuitGateChain Example

In the following example, a 3-qubit Quantum Fourier Transform is applied to an arbitrary input state

    using Qaintessent

    N = 3
    cgc = CircuitGateChain{N}(
        [
            single_qubit_circuit_gate(1, HadamardGate(), N),
            controlled_circuit_gate(2, 1, PhaseShiftGate(2π/2^2), N),
            controlled_circuit_gate(3, 1, PhaseShiftGate(2π/2^3), N),
            single_qubit_circuit_gate(2, HadamardGate(), N),
            controlled_circuit_gate(3, 1, PhaseShiftGate(2π/2^2), N),
            single_qubit_circuit_gate(2, HadamardGate(), N),
        ]
    )

Circuit

Circuit objects combine a CircuitGateChain and a MeasurementOps object. Applying a Circuit to a given input quantum state outputs the various expectation values from the measurement operators defined in the MeasurementOps object. A simple circuit is shown in the Circuit Example.

Qaintessent.Circuit — Type
Circuit{N}

Quantum circuit consisting of a unitary gate chain and measurement operators.

source
Qaintessent.apply — Method
apply(c::Circuit{N}, ψ::AbstractVector) where {N}

returns list of expectation values from measurement operators in c.meas after applying circuit gates in c.cgc on state vector of N qubits ψ

source

Circuit Example

    using Qaintessent
    N = 3
    cgc = CircuitGateChain{N}([
        single_qubit_circuit_gate(1, HadamardGate(), N),
        single_qubit_circuit_gate(2, HadamardGate(), N),
        single_qubit_circuit_gate(3, HadamardGate(), N),
    ])
    I = [1 0 ; 0 1]
    x = Qaintessent.matrix(XGate())
    meas_op = kron(kron(I,x), I)
    meas = MeasurementOps{N}([meas_op])
    c = Circuit{N}(cgc, meas)
